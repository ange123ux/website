More Details about programming languages

1. HTML

Hyper Text Markup language is the foundation and the wooden framework of the house, which doesn't look pretty unless CSS comes in. Some sites such as Ford, Peugeot and Lacoste use HTML.

2. CSS

Cascading Style Sheets is a collection of style tags used to make the website styled. It is taken as the paint to the house. Most websites make use of CSS.

3. Javascript

The simplest example of what Javascript can do is: If this, then that. As simple as that. Lost World Fairs as well as Cascade Brewery websites use Javascript.

4. Jquery

Jquery is a simpler version of Javascript since it saves individuals a lot of time and makes it much easier for them to add a feature that they like. Big companies like Google, Microsoft and IBM use jQuery.

5. Bootstrap

Bootstrap contributes in building responsive, mobile-first websites, and it is easy to learn it. Newsweek and Lyft sites use Bootstrap.
